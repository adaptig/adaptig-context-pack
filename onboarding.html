Claude Setup Guide

Get Claude working with Adaptig's context pack so your agent knows who we are, what we do, and how we communicate -- before you ask it anything.

1 Install Claude

Choose how you want to use Claude with context.

There are three Claude products you can use with the Adaptig context pack. All three work. Claude Code (the CLI) gives you the most control and is recommended for Adaptig work.

The CLI tool. Most powerful. Recommended for Adaptig work. It reads CLAUDE.md files automatically, runs in your terminal, and can read and write files directly.

Install it via npm, then run claude to authenticate with your Anthropic account.

npm install -g @anthropic-ai/claude-code

Then start it:

claude

It will open a browser window to authenticate. Follow the prompts.

Requires Node.js. If you do not have Node.js installed, download it from nodejs.org first.

The desktop app and web app (claude.ai). Best for conversational tasks like drafting, reviewing, and brainstorming.

Download the desktop app from claude.ai/download, or use claude.ai directly in your browser. Sign in with your Anthropic account.

The agentic mode for project-based work with file access. Available inside the Claude Desktop app.

Open a project, then enable Co-worker mode. It can read and write files in your project folder, making it well-suited for tasks that require editing documents or working across multiple files.

2 Create your work folder

A dedicated folder for Adaptig work.

Create a folder where your agent will look for context. This is where the context pack, future skills, and any Adaptig projects will live.

  1. Open Finder
  2. Press Cmd+Shift+H to go to your Home folder
  3. Press Cmd+Shift+N to create a new folder
  4. Name it Adaptig
  5. You now have a folder at ~/Adaptig
Home ▸ Adaptig
  1. Open File Explorer
  2. Navigate to your user folder (C:\Users\YourName)
  3. Right-click > New > Folder
  4. Name it Adaptig
mkdir -p ~/Adaptig
cd ~/Adaptig
This folder becomes the root of your Adaptig workspace. When you open Claude Code here, it automatically picks up the CLAUDE.md instruction file you will create in Step 4.

3 Clone the Context Pack

Pull the shared context files from the Adaptig GitHub org.

Prerequisites: Git installed, and your GitHub account has access to the adaptig GitHub organization. If you do not have access, ask on Roam or email sam@adaptig.com.

If you do not have Git installed, download GitHub Desktop from desktop.github.com -- it includes Git and provides a visual interface for cloning repositories.

Option A: SSH (if you have SSH keys configured)

cd ~/Adaptig
git clone git@github.com:adaptig/adaptig-context-pack.git

Option B: HTTPS (if you are unsure)

cd ~/Adaptig
git clone https://github.com/adaptig/adaptig-context-pack.git

Use SSH (Option A) if you have SSH keys configured with GitHub. Use HTTPS (Option B) if you are unsure -- it will prompt for your GitHub username and password.

The Context Pack contains eight markdown files with Adaptig's verified facts -- vision, values, products, brand voice, track record, and team. Your agent reads these before generating any Adaptig content.

What is in the pack

adaptig.md Company spine Start here
vision.md Vision, values, differentiators
product.md Spark / Shift / Shape / Show + adoption framework
achievement.md Track record numbers and reference clients
brand-tone-of-voice.md How Adaptig communicates
brand-visual-style.md Visual identity guidelines
team.md Core team and roles
README.md This guide + maintenance notes

4 Set up your CLAUDE.md

Tell Claude where to find the context pack.

CLAUDE.md is the instruction file Claude reads automatically when it starts. Create one in your Adaptig work folder so it knows to load the context pack.

Create a CLAUDE.md file in your ~/Adaptig folder with the content below. Choose whichever method you prefer:

Option A: Terminal

Run this command to create the instruction file:

cat > ~/Adaptig/CLAUDE.md << 'EOF'
# Adaptig Workspace

For Adaptig company context, read the files in ~/Adaptig/adaptig-context-pack/ before generating any Adaptig-facing content. Start with adaptig.md (the spine file) and read whichever other files are relevant to the task.
EOF
Option B: Manual (for non-terminal users)

Open any text editor (TextEdit on Mac, Notepad on Windows, or VS Code). Create a new file. Paste this content:

# Adaptig Workspace

For Adaptig company context, read the files in ~/Adaptig/adaptig-context-pack/ before generating any Adaptig-facing content. Start with adaptig.md (the spine file) and read whichever other files are relevant to the task.

Save the file as CLAUDE.md (not CLAUDE.md.txt) in your ~/Adaptig folder.

Next time you run claude from ~/Adaptig, it will read this file on startup and know where to find the context pack.

In Claude Desktop, create a new Project. Click the project settings icon. Under "Project knowledge", add the folder ~/Adaptig/adaptig-context-pack/.

Every conversation in this project will now have access to the context pack.

Open your Adaptig folder as a Co-worker project. Co-worker automatically reads CLAUDE.md from the project root.

If you created CLAUDE.md using the Claude Code tab instructions above, it already works. Otherwise, create the same CLAUDE.md file in ~/Adaptig/ using the content shown in the Claude Code tab (Option B).

5 Test your setup

Confirm Claude can find and use the context pack.

Open Claude Code in your Adaptig folder and run a few test prompts. If the context pack is loaded correctly, Claude should answer these without you providing any additional information.

cd ~/Adaptig
claude
Ask: "What is Adaptig's tagline?"
Expected: "AI adoption that sticks."
Ask: "What are Adaptig's four product lines?"
Expected: Spark (live workshops), Shift (AI Pioneers Program), Shape (executive training), and Show (keynote showcase).
Ask: "Draft a one-paragraph company intro for a proposal."
Expected: It should use the correct voice, cite 7,000+ participants, 180+ workshops, 4 continents, 8 languages, and position Adaptig as a practitioner-led training company -- without you having to feed it any of that.
If your agent answered correctly, you are set up.
