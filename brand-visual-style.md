# Brand Visual Style

Adaptig's visual identity is built around a felt-textured 3D illustration style, a primary wordmark with a standalone "A" logomark, and a six-color palette.

## Logo

The primary wordmark is available in multiple variants: with and without the `adaptig.ai` URL, on transparent and white backgrounds. A standalone "A" logomark is the companion symbol for favicons, avatars, and tight spaces. The official slogan asset is the "AI adoption that sticks" image. An Adobe Illustrator source file for the wordmark is maintained alongside the PNG variants.

## Typography

**Primary font: Proxima Nova.** The full family is available: standard, condensed, and extra-condensed widths, in weights from Thin through Black with italics. Usage notes and full specimen PDFs accompany the font distribution.

## Color palette

Six brand colors: **Green, Dark Blue, Beige, Light Blue, Black, Orange**. Each color is available as pre-rendered shape fills alongside the raw vector sources. Hex codes are available in the brand assets folder on the shared drive.

## Illustration style

**Felt-textured 3D characters** with soft lighting and warm materiality. Seventeen reference images define the canonical examples — any agent generating or commissioning Adaptig brand illustrations should match this style. The same style is used for Adaptig mascot animations.

**Image generation rule:** Use Nano Banana Pro (`gemini-3-pro-image-preview`), never Imagen 4. Pass the brand illustration reference images as references so the output matches the felt-textured house style.

## Shapes

Thirty branded geometric shape PNGs are available, rendered in each of the six palette colors, plus an Adobe Illustrator source file. Used as geometric elements in presentations, social posts, and marketing materials.

## Templates and assets

- **Presentation templates:** Multiple PowerPoint and Keynote files are available, along with an agency brand guide deck and the current canonical company presentation.
- **Email signatures:** HTML email signature templates exist with and without social-media blocks.
- **Brand intro video:** The latest Adaptig brand intro video is version 10. Version 9 is also available in six languages: English, German, Spanish, Chinese, Japanese, and Portuguese (Brazil).
- **Call backgrounds and LinkedIn banners:** Virtual meeting background images and LinkedIn profile and company page banners are maintained as separate asset sets.

## Where to find assets

Brand assets (logos, fonts, shapes, templates, call backgrounds, LinkedIn banners, email signatures) are stored on the Adaptig shared Google Drive under `Brand Assets/`. They are NOT included in this git repo — the repo is text-only context for AI agents.

If you need a brand asset for a deliverable, check the shared drive first — there is almost certainly an existing file for what you need. Never recreate from scratch when a canonical asset exists.
